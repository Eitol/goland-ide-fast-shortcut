##Shortcuts
Most shortcuts are triggered by shift + alt

####Window and tabs

shift+alt+6(Num KB)  => Split (move) right 

![a](imgs/move-r.gif)  

shift+alt+2(Num KB) => Split (move) down   

![a](imgs/move-d.gif)

shift+alt+(left/right) => Open left/right tab 

![a](imgs/move-tabs.gif) 




####Git

shift+alt+c => Push   
shift+alt+f => Fetch  
shift+alt+c => Commit
